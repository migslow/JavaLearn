final => constante, no puede tener mas clases descendientes, no se puede sobreescribir de sus descendientes
static => vale para todos
abstact => no puede crear objetos de esa clase, hay que implenentarlos si los quieres usar ( public abstract float importeNomina())
	   se puede implementar el constructor para que los hijos puedan hacer uso de los metodos
atributo instancia => son los propios (private String nombre)
